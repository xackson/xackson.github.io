Greetings

Greetings, friend. My name is Jack Kindel. I've built this website to act as a playground and a portfolio for my creative and technical antics. I believe one simple notion, and that is: after social media has invaded every inch of our lives, there will come a regression in terms of how we care to access each other online. This is why I am building a website.

Built with my own two hands, this website is running on GitHub, for free(mium), using GitHub Pages. It ain't much, but it's mine and it's working for now.
